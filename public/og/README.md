# OG Images for Agent Identity Cards

This directory contains static OG images for Twitter/X sharing.

## File Naming Convention

- `visionary.png` - The Visionary personality
- `explorer.png` - The Explorer personality
- `cultivator.png` - The Cultivator personality
- `optimizer.png` - The Optimizer personality
- `innovator.png` - The Innovator personality

## Image Specifications

- **Dimensions**: 1200 x 630 pixels
- **Format**: PNG (preferred) or JPG
- **Design**: Preview version of the agent identity card
- **Content**: Should match the carousel preview design on the website

## Usage

These images are served by `/api/og/agent/[agentUserId]` based on the agent's personality type.
